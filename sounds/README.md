# Sounds

Put custom sound files in this folder.

Supported filenames:

```text
play.wav / play.mp3 / play.m4a      - plays when selecting Play from the main menu
win.wav / win.mp3 / win.m4a         - plays when winning money
loss.wav / lose.wav / loss.mp3 ...  - plays when losing money
```

If a file is missing, the game continues silently.
