# Simple Python Roulette

A small terminal roulette game written in plain Python.

## Features

- Main menu with roulette, a future gamemode slot, and quit
- Starts you with a `$100` bankroll
- Supports bets on:
  - red / black
  - even / odd
  - exact number from `0` to `36`
- Uses standard roulette payouts:
  - even-money bets pay `1:1`
  - exact number bets pay `35:1`
- No external dependencies

## Run

```bash
python roulette.py
```

On some systems, use:

```bash
python3 roulette.py
```

## Note

This is just a simple game project, not real gambling.
